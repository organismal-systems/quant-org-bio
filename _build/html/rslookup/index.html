A key benefit of the Reynolds number is that it dramatically shrinks the number of experimental observations or numerical computations we need to characterize particle movement in fluids. That is because it enables us to make and interpret scale models. If we observe or compute one example of drag forces on an organism moving in fluid, the Reynolds number tells us how to translate that result to any geometrically similar organism in any fluid, as long as it has the same Reynolds number.

The Reynolds number rescues us from this difficult situation. We know that all geometrically similar organisms with the same Re\mathcal{Re}Re are scale models of each other (that is, they are dynamically similar).
